# Reference Files

This directory contains reference materials for the Jest + React Testing Library skill.

## Files

### query-cheatsheet.md

**Complete Query Methods Reference** - includes:
- All Testing Library query methods with full descriptions
- When to use each query with examples
- Detailed priority guide with performance considerations  
- Common ARIA roles list
- `getByRole` performance warning for large views

**When to use:** Need quick reference for specific query method or unsure which query to use.

### common-patterns.md

**Collection of Common Test Patterns** - includes:
- Form testing patterns (submission, validation, select/checkbox/radio)
- Async data loading tests with MSW
- Error handling tests
- Modal/Dialog tests
- List and pagination tests
- File upload, drag & drop tests
- Context/Provider and Custom Hooks tests

**When to use:** Need reference for specific scenario, can copy and adapt directly.
